- apa yang muncul di terminal jika kode ini di jalankan?

```js
function displayAddition() {
	console.log(10 + 2)
}
```

- apa yang muncul di terminal jika kode ini di jalankan?

```js
function displayAddition() {
	console.log(10 + 2)
}
displayAddition()
```

- apa yang muncul di terminal jika kode ini di jalankan?

```js
function displayAddition() {
	return 10 + 2
}
displayAddition()
```

- apa yang muncul di terminal jika kode ini di jalankan?

```js
function displayAddition() {
	return 10 + 2
}
console.log(displayAddition())
```

- apa yang muncul di terminal jika kode ini di jalankan?

```js
function displayAddition(num) {
	let total = num + 2
	return total + 2
}
console.log(displayAddition(5))
console.log(displayAddition(8))
```
